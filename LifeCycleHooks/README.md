# 🚀 Angular Lifecycle Hooks Visualizer

A simple and interactive Angular project to **understand lifecycle hooks visually** using **Bootstrap UI** and **color-coded logs**.

---

## 📌 Features

- ✅ Demonstrates all Angular lifecycle hooks  
- 🎨 Bootstrap-based clean UI  
- 📜 Real-time lifecycle logs panel  
- 🌈 Color-coded hook execution  
- 🔄 Update input to trigger change detection  
- ❌ Destroy/Create component dynamically  
- 🧠 Helps in interview preparation  

---

## 🧩 Lifecycle Hooks Covered

- Constructor  
- ngOnChanges  
- ngOnInit  
- ngDoCheck  
- ngAfterContentInit  
- ngAfterContentChecked  
- ngAfterViewInit  
- ngAfterViewChecked  
- ngOnDestroy  

---

## 🖥️ UI Preview

- Input data display  
- Scrollable lifecycle logs  
- Buttons to trigger lifecycle events  
- Clean card-based layout using Bootstrap
