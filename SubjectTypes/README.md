# 🚀 RxJS Subject Visualizer (Angular)

An interactive Angular project to understand the behavior of **Subject**, **BehaviorSubject**, and **ReplaySubject** using a clean UI and real-time logs.

---

## 📌 Features

- 🔵 Subject (No memory)
- 🟢 BehaviorSubject (Stores last value)
- 🟡 ReplaySubject (Stores last N values)
- 🎨 Bootstrap-based UI
- ⏱️ Timestamped logs
- 🔄 Reset functionality
- 🚫 Prevent duplicate subscriptions
- 📊 Easy visual comparison

---

## 🧠 Concepts Covered

- RxJS Observables
- Subject vs BehaviorSubject vs ReplaySubject
- Subscription handling
- Real-time data streaming
